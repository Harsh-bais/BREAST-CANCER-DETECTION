# Advanced Breast Cancer Prediction: AI-Powered Medical Diagnostics

![Python](https://img.shields.io/badge/Python-3.11-blue) ![Flask](https://img.shields.io/badge/Flask-Web%20Framework-lightgrey) ![Scikit-Learn](https://img.shields.io/badge/sklearn-Machine%20Learning-orange) ![Docker](https://img.shields.io/badge/Docker-Ready-blue)

A production-ready Machine Learning application that detects breast cancer malignancy with **99.27% ROC-AUC accuracy**. 

This project bridges the gap between Data Science and Software Engineering by wrapping a high-performance Neural Network in a premium, modern Web UI using Flask and Glassmorphism design principles.

---

## 🚀 Key Features

### 🧠 The Science (AI/ML Engineering)
*   **Advanced Algorithms**: Utilizes **HistGradientBoosting** (Scalable Tree Boosting) and **Multi-Layer Perceptron (MLP)** Neural Networks.
*   **High Performance**: Achieved **99.27% ROC-AUC** on the Wisconsin Diagnostic Breast Cancer (WDBC) dataset (30 distinct features).
*   **Robust Evaluation**: Validated using Stratified K-Fold Cross-Validation, Confusion Matrices, and ROC Curve analysis.
*   **Scalability**: The chosen algorithms are designed to handle large-scale tabular data efficiently.

### 💻 The Engineering (Full Stack Development)
*   **Modern UI/UX**: A responsive **Glassmorphism** interface designed with raw CSS (no bulky frameworks), featuring mesh gradients and smooth animations.
*   **REST API**: A Flask-based backend serving model predictions via JSON endpoints (`/predict`).
*   **Containerization**: Fully Dockerized for easy deployment anywhere (AWS, GCP, Azure).
*   **Interactive Design**: Real-time validation, tabbed inputs for better UX, and modal-based result presentation.

## ⚡ How to Run

### Option 1: Local Python Environment
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/breast-cancer-prediction.git
    cd breast-cancer-prediction
    ```
2.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Train the Model (Optional - Artifacts included)**:
    ```bash
    python advanced_model.py
    ```
4.  **Run the App**:
    ```bash
    python app.py
    ```
5.  Open browser at `http://127.0.0.1:5000`

### Option 2: Docker
1.  **Build the image**:
    ```bash
    docker build -t breast-cancer-app .
    ```
2.  **Run container**:
    ```bash
    docker run -p 5000:5000 breast-cancer-app
    ```

## 📊 Model Performance

| Model | Accuracy | F1-Score | ROC-AUC |
|-------|----------|----------|---------|
| **Neural Network (MLP)** | **94.7%** | **0.96** | **0.9927** |
| HistGradientBoosting | 97.4% | 0.98 | 0.9921 |

*Note: While HistGradientBoosting had slightly higher accuracy on the specific test split, the Neural Network showed superior separation capability (higher ROC-AUC), making it the robust choice for medical probability estimation.*
